# Error Handling

Error handling is used to handle errors before they occur and disrupt the exectution of code.

We use try and catch block to handle the errors.

## Try Block

The try block allows us to define a block of code which has to be tested for errors while its being executed.

```js
try{
    code;
}
```

## Catch Block

The catch block allows us to define a block of code to be executed when an error occured in the try block. It can only be executed when there is an try block.

```js
try {
    code;
} catch {
    code;
}
```
