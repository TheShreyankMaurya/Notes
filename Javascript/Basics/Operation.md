# Arithematic Operations

- Sum
  `let sum = a + b;`

- Difference
  `let sum = a - b;`

- Product
  `let sum = a * b;`

- Division
  `let sum = a / b;`

- Modulo(Returns Remainder)
  `let sum = a % b;`

- Exponential
  `let sum = a ** b;`

# Assignment Operators

- `(a = a + 1) == (a += 1);`

- `(a = a - 1) == (a -= 1);`

- `(a = a * 1) == (a *= 1);`

- `(a = a / 1) == (a /= 1);`

# Unary Operators

- Pre-Increment or Pre-Decrement(Change then Use)

    `++age, --marks`

- Post-Increment or Post-Decrement(Use then Change)

    `age++, marks--`

# Comparision Operators

- Less than
  `a < b`

- Less than equal to
  `a <= b`

- Greater than
  `a > b`

- Greater than equal to
  `a >= b`

- Compare only values
  `a == b`

- Compare values and types
  `a === b`

- Not equal to
  `a != b`
