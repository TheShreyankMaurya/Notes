# Loops

It is used to iterate a piece of code.

## Types of Loops

- for loop

    ```js
    for (initialisation; condition; updation) {
        code;
    }
    ```

- while loop

    ```js
    while (condition) {
        code;
        updation;
    }
    ```

- for of loop

    ```js
    for (element of collection) {
        code;
    }
    ```

- nested loops

    ```js
    for (initialisation, condition, updation){
        while (condition){
            code;
            updation;
        }
    }
    ```
