# Functions

Function are piece of code which will execute only when we call them.

## Types of functions

- Basic Function

```js
function fnName() {
    code;
}
```

- Function with arguments

```js
function fnName(arg1, arg2, arg3) {
    code;
}
```

- Return Function

```js
function fnName() {
    code;
    return val;
}
```

- Function Expression

```js
const fnName = function (arg1, arg2) {
    code;
};
```

- Arrow Function

```js
const fnName = (arg1, arg2) => {
    code;
};
```
