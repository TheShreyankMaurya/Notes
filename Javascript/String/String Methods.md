# String Methods

Strings are immutable means whenever we made some changes to the string it will always return a new string.

Common Syntax = strName.methodName()

- trim : trims whitespaces from both ends of the string.

- toUpperCase : convert all characters to uppercase.

- toLowerCase : convert all characters to lowercase.

- indexOf : returns the index of the element in the first occurence and it will return -1 if element not found.

- slice : returns a new string as a part of original string.
    - slice(start,end) : start from the starting index and ends at one character before ending index.

    - slice(start) : start from the starting index and goes till the end of the string.

    - slice(-ve n) : give n characters from the last of the string.

- replace(oldVal, newVal) : replace the value at the first occurence.

- replaceAll(oldVal, newVal) : replace all the values present in the string.

- repeat : use to repeat the value multiple times.
