# What is Python?

- Python is simple and easy.
- Free and open source.
- High level language.
- Develop by Guido van Rossum.
- Portable.

---

# Python character set

- Letters - A-Z, a-z.
- Digits - 0-9.
- Special symbols - + - \* /.
- Whitespaces - Blank space, tab, carriage return, newline, formfeed.
- Other Characters - Python can process all ASCII and Unicode characters as part of data or literals.

---

# Rules for identifiers

- Identifiers can be combination of uppercase, lowercase, digits or an underscore(\_).
- An identifier can not start with digit.
- We can't use any special symbols.
- Identifier can be any length.

---

# Types of Operators

An operator is a symbol that perform a certain operation on operands.

- Arithemetic Operator (+,-,\*,/,%,\*\*).
- Relational/Comparision Operator (==,!=,<=,>=,<,>).
- Assignment Operator (=,+=,-+,\*=,/=,%=,\*\*=).
- Logical Operator (and,or,not).
